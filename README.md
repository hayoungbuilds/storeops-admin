# StoreOps Admin

매장 운영을 위한 Admin Console (Orders / Settlement / Inventory / Settings).
2주 완성 목표로 “실서비스처럼 보이는 운영툴”을 빠르게 구축합니다.

## Preview
- Dashboard / Orders / Settlement / Inventory / Settings
- App Shell: fixed sidebar + topbar, viewport-based layout (main content scroll)
- Minimal UI tone inspired by modern fintech admin


---

## Tech Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS v4 + shadcn/ui
